# DecoStyle MERN + TypeScript Rewrite (Skeleton)

This is a **starter project structure** for migrating the PHP-based DecoStyle app
to a modern **MERN + TypeScript** stack.

## Structure

- `backend/` — Node.js + Express + TypeScript API (MongoDB via Mongoose)
- `frontend/` — React + TypeScript + Vite + Tailwind UI
- `docker/` — Basic Dockerfiles and docker-compose for local dev

All modules are just stubs. You will gradually port logic from the PHP version
into controllers, services, and React pages.
